# Examples of practical prompts for **Computer Science (CS) students**

**[Jim D. Pham](https://github.com/devservice-ora/articles/blob/main/README.md)**

To learn effectively, use AI to create **[concept maps](https://github.com/devservice-ora/articles/blob/main/MindTools%20-%20Concept%20Maps%20and%20Semantic%20Networks.pdf)** by asking a series of prompt questions about the topic in chronological order.

> I posed this question to AI: Give a list of prompt examples for CS students

---

### 1. The Code Debugger & Explainer

When your code isn't working, and you've been staring at it for an hour, you can use this to find the bug *and* learn why it happened.

* **Prompt Example:** *"I am getting a `NullPointerException` in this Java code. Don't just give me the corrected code; explain, line by line, why the error is happening and how to fix it. Here is my code: [Insert Code]"*

### 2. The Algorithmic Visualizer & Breakdown

CS theory can get incredibly abstract. This prompt helps you visualize how data moves through an algorithm step by step.

* **Prompt Example:** *"Explain how a Quicksort algorithm works using a small array of 6 numbers as an example. Show the state of the array after each pivot selection and swap so I can trace the logic step-by-step."*

### 3. The Code Optimization Partner

Writing code that works is step one; writing efficient code is step two. Use this to learn about Big O notation and performance.

* **Prompt Example:** *"Here is a Python function I wrote to find duplicates in a list. It currently runs in $O(n^2)$ time. Can you help me optimize it to run in $O(n)$ time using a hash set, and explain the difference in time and space complexity? Here is the code: [Insert Code]"*

### 4. The Pseudo-code Blueprint Creator

If you want to solve a programming assignment yourself but don't know where to start, ask for pseudo-code instead of actual code. This keeps you aligned with academic integrity while guiding your reasoning.

* **Prompt Example:** *"I need to write a C++ program that implements a basic Breadth-First Search (BFS) on a graph. Do not give me actual C++ code. Instead, give me a detailed pseudo-code blueprint and a list of the data structures (like queues) I will need to use."*

### 5. The Technical Interview Mock Coach

Perfect for practicing LeetCode-style questions for internships.

* **Prompt Example:** *"Act as a technical interviewer at a tech company. Ask me one standard Data Structures and Algorithms question appropriate for a sophomore CS student. Do not give me the answer. Wait for me to describe my approach and type out my solution, then critique my time complexity and edge cases."*

---

> **💡 CS Student Tip:** When pasting code into an AI, always use markdown code blocks (using triple backticks ```) to keep your indentation intact. This ensures the AI reads your Python loops or nested brackets perfectly!
